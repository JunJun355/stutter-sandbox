# stutter-sandbox

Experiments are organized by directory:

- `exp1_ip_collisions/`  
  DNS IP-collision measurement and reporting.  
  See `exp1_ip_collisions/README.md`.
- `exp2_dns_middleman/`  
  Local DNS middleman and macOS DNS routing helpers.  
  See `exp2_dns_middleman/README.md`.
- `exp3_no_youtube/`  
  Browser tooling for DNS-path testing (DoH-disabled Chromium launch).  
  See `exp3_no_youtube/README.md`.
